- Spring Security Basics
- HTML, CSS and Bootstrap UI Design
- JavaScript for Client-side Functionality
- CRUD Operations
- Client-Server Communication
- Validation and Exception Handling
- Maven Project Management
- Git and GitHub Version Control

This project significantly improved my understanding of software architecture, backend development, API communication, and responsive user interface design.
